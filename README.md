# Single-Page-CV
Build a Single-page CV purely using HTML5

## Commonly Used Tags
+ Header Tag: for providing heading from h1 to h6
+ Paragraph Tag
+ Anchor Tag: to add link
+ Image Tag: to add image to webpage
+ Small Tag: to keep text size smaller
+ Unordered Tag: to build unordered list
+ Bold Tag: bold the text
+ Break Line Tag: to proive space or add new line
+ Pre Tag: displays text as it is(without ignoring spaces and next line)

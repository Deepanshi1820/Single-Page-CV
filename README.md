# Single-Page-CV
Build a Single-page CV purely using HTML5

## Commonly Used Tags
+ Header Tag: for providing heading from h1 to h6
+ Paragraph Tag(<p></p>)
+ Anchor Tag(<a></a>): to add link
+ Image Tag(<img>): to add image to webpage
+ Small Tag(<small>): to keep text size smaller
+ Unordered Tag(<ul>): to build unordered list
+ Bold Tag(<b></b>): to bold the text
+ Break Line Tag(<br>)
+ Pre Tag(<pre></pre>): to obtain text as it is without any changes
